# Movie Rating Prediction with Python

## Project Overview
This project predicts movie ratings using Machine Learning using the IMDb Movies India dataset.

## Dataset
IMDb Movies India Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm
- Linear Regression

## Steps
- Data Cleaning
- Data Preprocessing
- Label Encoding
- Feature Selection
- Train-Test Split
- Model Training
- Model Evaluation

## Results
- MAE: 1.03
- RMSE: 1.29
- R² Score: 0.10
